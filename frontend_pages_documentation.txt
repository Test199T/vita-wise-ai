Frontend Pages Documentation - Vita Wise AI
===============================================

ภาพรวม
------
แอปพลิเคชัน Vita Wise AI เป็นแพลตฟอร์มการจัดการสุขภาพและความเป็นอยู่ที่ดีที่ใช้ AI ช่วยวิเคราะห์และให้คำแนะนำ แอปพลิเคชันมีหน้าต่างๆ รวม 15 หน้า ที่ครอบคลุมฟีเจอร์ครบถ้วนสำหรับการดูแลสุขภาพแบบครบวงจร

สรุปหน้าต่างๆ (Quick Overview)
==============================

หน้าหลัก (2 หน้า)
-----------------
- Index.tsx - หน้าแรก/หน้าหลัก
- Dashboard.tsx - แดชบอร์ดหลัก

ล็อกอิน/ลงทะเบียน (2 หน้า)
---------------------------
- Login.tsx - เข้าสู่ระบบ
- Register.tsx - ลงทะเบียน

บันทึกสุขภาพ (4 หน้า)
---------------------
- FoodLog.tsx - บันทึกอาหาร
- WaterLog.tsx - บันทึกน้ำดื่ม
- ExerciseLog.tsx - บันทึกออกกำลังกาย
- SleepLog.tsx - บันทึกการนอน

จัดการสุขภาพ (2 หน้า)
---------------------
- HealthGoals.tsx - เป้าหมายสุขภาพ
- Profile.tsx - โปรไฟล์ผู้ใช้

ความช่วยเหลือ (2 หน้า)
-----------------------
- AIInsights.tsx - ข้อมูลจาก AI
- Chat.tsx - แชทและช่วยเหลือ

อื่นๆ (3 หน้า)
--------------
- Onboarding.tsx - แนะนำการใช้งาน
- Notifications.tsx - การแจ้งเตือน
- NotFound.tsx - หน้า 404

สถิติรวม
--------
- จำนวนหน้าทั้งหมด: 15 หน้า
- ขนาดไฟล์รวม: ประมาณ 350KB
- จำนวนบรรทัดรวม: ประมาณ 6,000+ บรรทัด
- ระดับความซับซ้อนเฉลี่ย: ⭐⭐⭐⭐☆

สถาปัตยกรรม
-----------
- Frontend Framework: React + TypeScript
- UI Library: Shadcn/ui + Tailwind CSS
- State Management: React Context + Hooks
- Routing: React Router
- Responsive Design: Mobile-First Approach

Design System
-------------
- Color Scheme: Health-focused colors
- Typography: Modern, readable fonts
- Icons: Consistent icon set
- Components: Reusable UI components
- Layout: Grid-based responsive layout
